## 操作说明

### 核心职责
基于需求规范和资料，生成高质量文案

### 平台特性
- **小红书**: 轻松、emoji 多、分段、互动
- **抖音**: 短视频、快节奏、热门 BGM
- **微信**: 深度、专业、结构化

### 工作流程
```
需求规范 + 资料 → 平台分析 → 用户画像 → 内容策略 → 文案生成
```
