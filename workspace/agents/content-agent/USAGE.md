# Content Agent 使用指南

**版本**: v3.0
**更新时间**: 2026-03-05 13:45 UTC

---

## 🎯 功能概述

Content Agent 是一个智能文案生成工具，能够为不同平台（小红书、抖音、微信、B站）生成高质量、平台适配的文案。

---

## 🚀 快速开始

### 1. 基础生成（v2.0）

快速生成基于模板的文案，无需 API 配置。

```bash
cd /home/node/.openclaw/agents/content-agent

# 示例1: 小红书文案
bash scripts/generate-v2.sh "小红书" "AI工具推荐" "轻松"

# 示例2: 抖音文案
bash scripts/generate-v2.sh "抖音" "ChatGPT技巧" "快节奏"

# 示例3: 微信文案
bash scripts/generate-v2.sh "微信" "AI行业趋势" "专业"
```

**输出位置**: `output/content_YYYYMMDD_HHMMSS.md`

### 2. 智能生成（v3.0）

集成 GLM-4-Plus，自动生成高质量完整文案。

```bash
# 示例1: 小红书文案（智能）
bash scripts/generate-v3.sh "小红书" "AI工具推荐" "轻松"

# 示例2: 抖音文案（智能+背景）
bash scripts/generate-v3.sh "抖音" "ChatGPT技巧" "快节奏" "针对新手用户"

# 示例3: 微信文案（智能+深度）
bash scripts/generate-v3.sh "微信" "AI行业趋势" "专业" "2024年最新趋势分析"
```

**当前状态**: ⚠️ 需要配置 GLM-4-Plus API

## 📋 支持的平台

### 小红书
- **用户群体**: 18-35岁女性为主
- **内容风格**: 真实体验、实用干货、闺蜜对闺蜜
- **Emoji使用**: 大量使用
- **标题特点**: emoji+短句+感叹号
- **结构**: 开场引入+3-5个要点+总结+话题标签

### 抖音
- **用户群体**: 18-30岁，性别均衡
- **内容风格**: 快节奏、强感染力、3秒抓住注意力
- **Emoji使用**: 适度使用
- **标题特点**: 短平快，数字+emoji
- **结构**: 3秒hook+核心内容+总结+话题标签

### 微信
- **用户群体**: 25-45岁职场人群
- **内容风格**: 深度内容、专业见解、结构化
- **Emoji使用**: 少量使用
- **标题特点**: 专业、准确
- **结构**: 引言+背景+核心分析+总结

### B站
- **用户群体**: 18-25岁，男性略多
- **内容风格**: 有趣、有料、有梗
- **Emoji使用**: 大量使用
- **标题特点**: 年轻化、网络热词
- **结构**: 开场引入+干货内容+互动引导

## ⚙️ 配置说明

### v2.0（基础版本）
✅ 无需配置，直接使用

### v3.0（智能版本）
⚠️ 需要配置 GLM-4-Plus API

**配置步骤**:
1. 获取 GLM-4-Plus API Key
2. 修改 `scripts/generate-v3.sh` 中的 API 调用部分
3. 取消注释 API 调用代码
4. 重新运行

**API 配置示例**:
```bash
# 在 generate-v3.sh 中找到并修改
API_KEY="your-api-key-here"

# 取消注释这部分代码
curl -s -X POST https://open.bigmodel.cn/api/anthropic/v1/messages \
  -H "Authorization: Bearer $API_KEY" \
  ...
```

## 🔧 高级用法

### 自定义背景信息
```bash
bash scripts/generate-v3.sh "平台" "主题" "风格" "背景信息"
```

**示例**:
```bash
# 小红书文案，针对新手
bash scripts/generate-v3.sh "小红书" "AI工具推荐" "轻松" "针对完全没有AI基础的新手用户"

# 微信文案，加入最新数据
bash scripts/generate-v3.sh "微信" "AI行业趋势" "专业" "参考2024年第一季度市场报告"
```

### 批量生成
```bash
# 创建任务列表
cat > tasks.txt << EOF
小红书|AI工具推荐|轻松
抖音|ChatGPT技巧|快节奏
微信|AI行业趋势|专业
EOF

# 批量执行
while IFS='|' read -r platform topic style; do
  bash scripts/generate-v2.sh "$platform" "$topic" "$style"
done < tasks.txt
```

## 📊 质量标准

### 好文案的标准
- ✅ **标题吸引力**: 3秒内抓住注意力
- ✅ **Hook 相关性**: 开场与主题高度相关
- ✅ **内容完整性**: 提供有价值的信息
- ✅ **平台适配性**: 符合平台用户习惯
- ✅ **CTA 清晰度**: 明确的行动召唤

### 避免的错误
- ❌ 跨平台复制粘贴
- ❌ 标题党（不兑现承诺）
- ❌ 内容空洞（无实际价值）
- ❌ 忽略平台特性
- ❌ 缺少互动引导

---

## 💡 最佳实践

1. **明确目标受众**: 在背景信息中详细描述
2. **提供具体数据**: 让内容更有说服力
3. **使用真实案例**: 增加可信度
4. **添加互动引导**: 鼓励评论、点赞、收藏
5. **测试优化**: 尝试不同标题和Hook，选择效果最好的

## 🆘 故障排除

### 问题1: 生成的文案质量不高
**解决方案**:
- 使用 v3.0（智能版本）
- 提供详细的背景信息
- 明确目标受众和需求

### 问题2: API 调用失败
**解决方案**:
- 检查 API Key 是否正确
- 检查网络连接
- 暂时使用 v2.0（基础版本）

### 问题3: 平台特性不明显
**解决方案**:
- 明确指定平台
- 查看平台分析部分
- 根据平台特性调整
