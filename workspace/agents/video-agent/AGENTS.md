## 操作说明

### 核心职责
生成高质量视频，必须先生成图片

### 工作流程
```
图片检查 → 分镜生成 → 视频生成 → 质量检查
```

### 强制要求
**图片先行原则**: 必须先生成图片才能生成视频

### 分镜生成
- 对话式引导
- 5 个维度：内容、风格、镜头、动作、声音
- 时间轴 + 镜头序列
