# 批量内容生产系统 - 使用指南

## 🎯 核心功能

根据自然语言描述，批量生成多角度、多形式的内容。

---

## 📖 使用方法

### 基础用法

```
批量生成 5 篇关于AI工具的文案，包括职场新人、程序员、学生党角度
```

### 指定形式

```
批量生成 AI工具推荐内容，包括图文、视频、文章三种形式，角度为职场新人、程序员
```

### 完整指定

```
批量生成 10 篇ChatGPT使用技巧的内容：
- 角度：职场新人、程序员、学生党、自由职业、企业主
- 形式：图文、视频、文章
- 平台：小红书、抖音、微信
```

---

## 🔍 支持的角度

- **职场新人** - 友好调性，150-200字
- **程序员** - 专业调性，1000-1500字
- **学生党** - 轻松调性，150-200字
- **自由职业** - 活泼调性，200-300字
- **企业主** - 权威调性，1500-2000字

---

## 🎨 支持的形式

### 图文（小红书）
- 长度: 150-200字
- 格式: emoji + 标签
- 视觉: fresh/list/warm

### 视频（抖音）
- 长度: 60秒
- 格式: 口播文案 + 节奏
- 视觉: bold/vivid

### 文章（微信）
- 镾度: 1000-1500字
- 格式: 长文 + 排版
- 视觉: minimal/pyramid

---

## 🔄 执行流程

```
自然语言输入
  ↓
解析配置
  ↓
生成任务列表
  ↓
并行执行:
  ├─ content-producer × N (生成文案)
  ├─ visual-designer × N (生成视觉)
  ├─ video-producer × N (生成视频)
  └─ quality-reviewer × N (质量审核)
  ↓
整合结果
  ↓
返回批量报告
```

## 🚀 高级用法

```

```

### 多平台分发


### 定时执行（cron）

```bash
# 每天 9:00 批量生成
0 9 * * * /path/to/batch-producer.sh "AI工具推荐"

# 每 4 小时生成一批
0 */4 * * * /path/to/batch-producer.sh "ChatGPT技巧"
```

## 🎯 关键优势

### 1. 真正的并行执行

- content-producer × 5 (同时生成5个角度)
- visual-designer × 5 (同时生成5个视觉)
- quality-reviewer × 5 (同时审核5个内容)

### 2. 快速批量产出

- 5个角度 × 3种形式 = 15个内容
- 并行执行，节省时间
- 预计 3-5 分钟完成

### 3. 自然语言控制

- 无需复杂配置
- 用自然语言描述需求
- 自动解析和执行

### 4. 质量保证

- 每个内容都经过质量审核
- 不合格内容自动标记
- 提供改进建议

## 🔧 故障排查

### 问题 1: sessions_send 未定义

**症状**: 提示 sessions_send is not defined

**原因**: 不在 Agent 环境中

**解决**: 确保在 Agent session 中调用

### 问题 2: Agent 不响应

**症状**: 某个 Agent 长时间无响应

**原因**: 任务太复杂或网络问题

**解决**: 
- 增加超时时间
- 简化任务描述
- 检查网络连接

### 问题 3: 结果格式错误

**症状**: Agent 返回格式不符合预期

**解决**:
- 在任务中明确要求格式
- 添加格式验证
- 提供示例

---

## 🎓 最佳实践

1. **明确需求**: 输入越具体，输出越准确
2. **合理预期**: 批量生成需要时间
3. **检查结果**: 查看质量评分，必要时调整
4. **反馈优化**: 根据实际效果优化配置
