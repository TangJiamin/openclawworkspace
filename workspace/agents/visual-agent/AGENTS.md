# visual-agent - 操作指南

**版本**: v3.1 - 精简版
**更新**: 2026-03-09

---

## ⚠️ 最高优先级

1. **AI 模型决策** - 智能选择技术路线
2. **质量第一** - 图片质量 ≥ 85 分
3. **图片先行** - 视频生成必须先有图片

---

## 🎨 图片质量标准

**小红书评分**（≥ 85 分）:
- 清晰度 (30%): ≥ 1080p
- 风格匹配 (30%): ≥ 90 分
- 尺寸比例 (20%): 3:4
- 色彩搭配 (20%): ≥ 85 分
- AI 科技感 (20%): ≥ 85 分

---

## 🛠️ 技术路线

| 工具 | 用途 | 命令 |
|------|------|------|
| **Refly Canvas** | 可视化预览 | 对话式使用 |
| **visual-generator** ⭐ | 智能生成 | 自动选择模型 |
| **Xskill API** | 直接调用 | `bash xskill_call.sh <model> <prompt>` |

**AI 决策**:
- 可视化预览 → Refly Canvas
- 日常创作 → visual-generator（jimeng-5.0）
- 商业项目 → visual-generator（flux-realism）
- 视频生成 → visual-generator（seedance-pro）

---

## 🎯 visual-generator 智能路由

```javascript
// AI 模型决策
if (场景 === "小红书") {
  模型 = "jimeng-5.0"
} else if (场景 === "商业") {
  模型 = "flux-realism"
} else if (场景 === "视频") {
  模型 = "seedance-pro"
} else {
  模型 = "jimeng-5.0" // 默认
}
```

## 🔄 调用流程

```
用户需求
  ↓
AI 模型分析（场景识别）
  ↓

  ↓

  ↓
输出结果
```

## 📝 质量审核

**评分标准**:
- 综合评分 ≥ 85 分

**不及格**: 重新生成（最多 3 次）
