## 🎯 使用原则

### 1. 永远不在脚本中硬编码

❌ **错误**:
```bash
export API_KEY="sk-xxx"
```

✅ **正确**:
```bash
# 从 .env 文件读取
source /home/node/.openclaw/workspace/.env
```

### 2. 配置文件使用引用

❌ **错误**:
```json
{
  "apiKey": "sk-xxx"
}
```

✅ **正确**:
```json
{
  "apiKey": "${XSKILL_API_KEY}"
}
```

### 3. 查找顺序

当需要使用 API Key 时：
1. 首先检查环境变量
2. 如果环境变量不存在，提示用户设置
3. 永远不在代码中硬编码
