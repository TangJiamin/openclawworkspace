## 关键发现

### 1. 官方 vs 社区 Skills 格式

**官方格式**（YAML Frontmatter）:
```yaml
---
name: weather
description: "..."
homepage: https://...
metadata: { ... }
---
```

**社区格式**（AgentSkills）:
```yaml
location: /path/to/SKILL.md
```

**结论**: 两种格式都是正确的，不需要统一

### 2. 工具调用最佳实践

**错误**:
```python
task = "运行命令：bash xxx.sh"
```

**正确**:
```python
task = "生成一张图片"
```

**原则**: 描述需求，而不是命令
