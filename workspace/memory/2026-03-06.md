**问题**: visual-generator 输出参数标签（style=fresh），jimeng-5.0-api 需要自然语言提示词

**最优解**: 在 visual-generator 中添加参数→提示词转换逻辑

**实现**:
1. ✅ 创建 `jimeng-5.0-api` Skill
   - `scripts/config.sh` - API 配置
   - `scripts/generate.sh` - 文生图 + 图生图
   - 支持轮询任务状态

2. ✅ 在 `visual-generator` 中添加提示词转换
   - `scripts/params_to_prompt.sh` - 参数→提示词映射
   - `scripts/generate.sh` - 主生成脚本（v2.1）
   - 输出: PROMPT + PARAMS (JSON)

3. ✅ 更新 `visual-agent/AGENTS.md`
   - 技术路线优先级调整（jimeng-5.0-api 提升至第2位）
   - 新增调用流程示例
   - 版本更新至 v3.1

### 架构设计

```
visual-agent (AI 决策)
  ↓
visual-generator (参数推荐 + 提示词生成)
  ├─ params_to_prompt.sh ⭐ 核心逻辑
  └─ 输出: PROMPT + PARAMS
  ↓
jimeng-5.0-api (接收 PROMPT)
  ↓
生成图片 ✅
```

### 关键洞察

1. **参数转换是关键**: visual-generator 的参数标签需要转换为自然语言才能被 AI 图像模型理解
2. **职责分离**: visual-generator 负责参数+提示词，jimeng-5.0-api 只负责 API 调用
3. **渐进式加载**: 两个 Skills 可以独立加载，节省 Token

## 🎓 定向学习：Agent 协同工作流（2026-03-06 21:31）

### 学习主题
深度理解 Agent 协同工作流和质量审核标准

### 学习成果

#### 1. Agent 协同模式 ⭐⭐⭐⭐⭐

**串行执行**（依赖关系）:
```
visual-agent → video-agent（必须先有图片）
```

**并行执行**（无依赖）:
```javascript
Promise.all([
  requirement-agent,
  research-agent
])
```

**智能调度**（根据关键词）:
| 关键词 | 编排策略 |
|--------|---------|
| "视频" | 强制流程：visual-agent → video-agent |
| "生成"/"创作" | 完整流程：requirement → research → content → visual → video → quality |
| "收集"/"搜索" | 简化流程：requirement → research |

#### 2. 视频生成流程 ⭐⭐⭐⭐⭐

**seedance-storyboard 核心价值**:
- ✅ 生成分镜提示词（不是直接生成视频）
- ✅ 将用户模糊想法转化为专业提示词
- ✅ 符合 Seedance 2.0 语法

**标准分镜结构**（15秒）:
```
0-3秒：开场镜头（远景/全景）
3-6秒：发展镜头（中景/近景）
6-10秒：高潮镜头（近景/特写）
10-13秒：转折镜头（过渡）
13-15秒：结尾镜头（全景/远景）
```

#### 3. 质量审核标准 ⭐⭐⭐⭐⭐

**视频质量审核维度**:
- **流畅度** (30%) - 视频流畅，无卡顿
- **节奏感** (30%) - 节奏合适，快慢适中
- **时长匹配度** (25%) - 时长符合要求
- **视觉吸引力** (15%) - 有吸引力，视觉效果好

**整体评分**:
```
整体评分 = (需求质量 × 15%) + (资料质量 × 20%) + (文案质量 × 25%) + (图片质量 × 20%) + (视频质量 × 20%)
```
