### 关键发现

**职责**:

#### 2. 分镜提示词结构（学习自 SKILL.md v3.0）

**标准结构**（15秒）:
```
0-3 秒：开场镜头（远景/全景）
3-6 秒：发展镜头（中景/近景）
6-10 秒：高潮镜头（近景/特写）
10-13 秒：转折镜头（过渡）
13-15 秒：结尾镜头（全景/远景）
```

**提示词格式**:
```
电影级写实风格，15秒，16:9宽屏，日落黄金时刻

0-3秒：远景缓慢推近，海平线夕阳，女孩剪影
3-7秒：中景环绕镜头，女孩开始旋转
7-11秒：近景跟随移动，女孩面向镜头
11-13秒：特写手部动作，手指划过夕阳
13-15秒：远景拉远，女孩在落日余晖中定格
```

## Step 5: 质量审核标准分析

### quality-agent 的评分体系

**各阶段质量阈值**:
- requirement_review: 80/100
- research_review: 80/100
- content_review: 85/100
- visual_review: 85/100
- video_review: 85/100

**视频质量审核维度**（video_review）:
- **流畅度** (30%) - 视频流畅，无卡顿
- **节奏感** (30%) - 节奏合适，快慢适中
- **时长匹配度** (25%) - 时长符合要求
- **视觉吸引力** (15%) - 有吸引力，视觉效果好

**整体评分计算**:
```
整体评分 = (需求质量 × 15%) + (资料质量 × 20%) + (文案质量 × 25%) + (图片质量 × 20%) + (视频质量 × 20%)
```

### 评级标准

| 整体评分 | 评级 | 说明 |
|---------|------|------|
| ≥ 90 | 优秀 | 质量很高，可以直接使用 |
| 85-89 | 良好 | 质量较好，可以微调使用 |
| 80-84 | 及格 | 质量一般，建议改进后使用 |
| < 80 | 不及格 | 质量较差，需要重新生成 |

## Step 6: 关键洞察

### 1. 第一性原理分析

**问题**: seedance-storyboard 需要图片输入

**表象思维**: "seedance-storyboard 不完善"

**第一性原理**:
- 本质: seedance-storyboard 是图生视频工具，需要图片作为参考
- 拆解: 两个模式（图片输入 / 纯文本提示词）
- 最优解: 完善纯文本提示词模式，支持无图片输入

### 2. 职责分离原则

**清晰的职责划分**:
```
requirement-agent: 需求分析
  ↓
research-agent: 资料收集
  ↓
content-agent: 内容生成
  ↓
visual-agent: 图片生成
  ↓
video-agent: 视频生成（调用 seedance-storyboard）
  ↓
quality-agent: 质量审核
```

**关键原则**:
- ✅ 每个 Agent 专注于自己的职责
- ✅ 不要越界（例如：seedance-storyboard 不应该直接生成视频）
- ✅ 通过 AI 模型决策技术路线

### 3. AI 模型决策 > 固定规则

**传统思维**（固定规则）:
```bash
if duration < 10:
  use "jimeng-video-api"
else:
  use "seedance-pro-api"
```

**AI 模型决策**（智能判断）:
```
video-agent 的 AI 模型分析：
- 用户需求：质量优先？
- 平台特性：抖音/小红书？
- 预算限制：商业项目/个人创作？

→ 智能选择技术路线
```
