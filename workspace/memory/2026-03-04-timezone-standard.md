## ⏰ 时间换算公式

```
中国时间 = UTC + 8 小时
UTC 时间 = 中国时间 - 8 小时
```

### 示例

| 中国时间 | UTC 时间 |
|---------|---------|
| 03:00 (凌晨) | 前一天 19:00 |
| 10:00 (上午) | 02:00 |
| 14:00 (下午) | 06:00 |
| 22:00 (晚上) | 14:00 |

## 🎯 配置规范

### 创建新定时任务时

**必须指定**:
```bash
--tz "Asia/Shanghai"
```

**示例**:
```bash
openclaw cron add \
  --name "任务名称" \
  --cron "0 3 * * *" \
  --tz "Asia/Shanghai" \
  ...
```
