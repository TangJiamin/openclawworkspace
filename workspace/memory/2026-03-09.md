#### 3. 质量审核标准 ⭐⭐⭐⭐⭐

**视频质量审核维度**:
- **流畅度** (30%): 视频流畅，无卡顿
- **节奏感** (30%): 节奏合适，快慢适中
- **时长匹配度** (25%): 时长符合要求
- **视觉吸引力** (15%): 有吸引力，视觉效果好

**整体评分计算**:
```
整体评分 = (需求质量 × 15%) + (资料质量 × 20%) + (文案质量 × 25%) + (图片质量 × 20%) + (视频质量 × 20%)
```

**评级标准**:
| 整体评分 | 评级 | 说明 |
|---------|------|------|
| ≥ 90 | 优秀 | 质量很高，可以直接使用 |
| 85-89 | 良好 | 质量较好，可以微调使用 |
| 80-84 | 及格 | 质量一般，建议改进后使用 |
| < 80 | 不及格 | 质量较较差，需要重新生成 |
