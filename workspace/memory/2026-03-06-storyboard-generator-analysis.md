### 分镜表生成器的独特之处

#### 1. 完整的工作流程设计

**5 个步骤**:
1. 分析剧本结构
2. 导演三帧分镜（起帧/爆帧/钩帧）
3. 展开完整镜头表
4. 生成提示词
5. QC 检查清单

#### 2. 导演三帧法

**每场生成 3 张关键帧**：
- **起帧** (KF_A)：建立环境与关系
- **爆帧** (KF_B)：冲突/动作爆点
- **钩帧** (KF_C)：悬念/转场/情绪余波

#### 3. 提示词三段式

**强制格式**:
```
最终提示词 = 风格段 + 主体段 + 画面段

┌──────────┐   ┌──────────┐   ┌──────────┐
│ 风格段    │ + │ 主体段    │ + │画面段    │
└──────────┘   └────┬─────┘   └────┬─────┘
     │              │              │
     ▼              ▼              ▼
┌──────────┐   ┌──────────┐   ┌──────────┐
│STYLE_BASE│   │@角色ID   │   │SceneKey  │
│风格一句话│   │形象描述  │   │场景描述  │
│          │   │+表情动作 │   │+景别机位 │
│          │   │增量      │   │+定格瞬间 │
└──────────┘   └──────────┘   └──────────┘
```

#### 4. 详细的决策标准

**场景匹配标准**:
```
| 景别 | 用途 | 示例场景 |
|------|------|----------|
| 远景 | 建立环境、展示规模 | 山川全貌、城池远眺 |
| 全景 | 展示人物全身、空间关系 | 角色站立、多人场景 |
| 中景 | 屓示动作、人物互动 | 对话、交手 |
| 近景 | 展示表情、情绪 | 反应、独白 |
| 特写 | 强调细节、关键道具 | 眼神、手部、道具 |
```

**机位选择标准**:
```
| 机位 | 情绪效果 | 示例场景 |
|------|----------|----------|
| 平视 | 中性、客观 | 日常对话 |
| 俯视 | 弱势、渺小 | 角色陷入困境 |
| 仰视 | 强势、威严 | 强者出场、压迫感 |
| 越肩 | 代入、对峙 | 两人对话 |
| 主观 | 沉浸、紧张 | 发现敌人、探索 |
```

#### 5. 质量检查体系

**三级检查**:
- P0（必须修复）：影响剪辑
- P1（第二优先）：影响一致性
- P2（最后修复）：美观问题
