# 开发工具配置

**更新时间**: 2026-03-04 02:45 UTC (中国时间 10:45)

## ⚠️ 重要原则

**Python 相关程序使用 uv，Node.js 相关程序使用 bun**

---

## 📦 已安装工具

### 1. uv - Python 包管理器

**版本**: 0.10.8  
**安装位置**: ~/.local/bin/uv  
**官方地址**: https://github.com/astral-sh/uv

**使用方式**:
```bash
# 运行 Python 脚本
uv run script.py

# 安装包
uv pip install package-name

# 创建虚拟环境
uv venv
```

### 2. bun - JavaScript 运行时和包管理器

**版本**: 1.3.10  
**安装位置**: ~/.local/bin/bun  
**官方地址**: https://bun.sh

**使用方式**:
```bash
# 运行 JavaScript/TypeScript
bun run script.ts

# 安装包
bun install package-name

# 运行脚本
bun script.js
```

## 🔄 替代方案

### Python 旧方式 → 新方式

| 旧方式 | 新方式 |
|--------|--------|
| `python script.py` | `uv run script.py` |
| `pip install` | `uv pip install` |
| `python3 -m venv` | `uv venv` |

### Node.js 旧方式 → 新方式

| 旧方式 | 新方式 |
|--------|--------|
| `npm install` | `bun install` |
| `node script.js` | `bun run script.js` |
| `npx command` | `bunx command` |
