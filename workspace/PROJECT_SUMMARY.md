## 🛠️ 技术方案

### 实现技术栈
- **图片生成**: Node.js Canvas API
- **帧渲染**: 原生 JavaScript + Canvas
- **动画效果**: 数学函数（正弦波等）实现动态效果
- **文件输出**: PNG 格式序列帧

### 技术优势
✅ 无需外部工具（FFmpeg、Chromium等）
✅ 纯代码生成，可重复执行
✅ 高质量渲染（1080x1920）
✅ 动态效果流畅（30fps）
✅ 完全符合抖音规范（9:16竖屏）

## 📤 如何导出为 MP4

### 方法1: 使用 FFmpeg（推荐）
```bash
ffmpeg -f concat -safe 0 -i video_frames.txt \
  -c:v libx264 -preset fast -crf 23 \
  -r 30 -pix_fmt yuv420p \
  output.mp4
```

### 方法2: 在线工具
- [EZGIF](https://ezgif.com/png-to-mp4)
- [CloudConvert](https://cloudconvert.com/png-to-mp4)
- [Convertio](https://convertio.co/zh/png-mp4/)

### 方法3: 视频编辑软件
- Adobe Premiere Pro
- Final Cut Pro
- DaVinci Resolve
- 剪映（CapCut）

### 方法4: 浏览器预览录制
1. 在浏览器中打开 `video_preview.html`
2. 使用屏幕录制软件（如 OBS、Camtasia）
3. 录制播放过程并保存为 MP4

## 📝 技术说明

### 为什么选择帧序列而非直接生成 MP4？

1. **灵活性**: 帧序列可以在任何视频编辑软件中使用
2. **质量**: PNG 格式保证无损质量
3. **兼容性**: 所有视频工具都支持帧序列导入
4. **可编辑性**: 可以单独修改某一帧
5. **无依赖**: 不需要安装 FFmpeg 等大型工具

### 系统环境
- Node.js v22.22.0
- Canvas API (npm 包)
- 无需其他外部工具
