## 🔧 修复方案

### 问题1: JSON 文件格式

**当前**:
```bash
echo "[]" > "$BATCH_CONTENT_RESULTS"
echo "$RESULT" >> "$BATCH_CONTENT_RESULTS"  # ❌ 错误：不是有效的 JSON 数组
```

**修复**:
```bash
# 使用 jq 正确构建 JSON 数组
echo "$RESULT" | jq --slurpfile results "$BATCH_CONTENT_RESULTS" '. + $results' > "$BATCH_CONTENT_RESULTS.$$"
```
