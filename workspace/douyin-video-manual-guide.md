**比例**: 9:16 竖版（1080×1920）

## 📋 完整分镜脚本

### 场景1: 开头（0-3秒）

**画面**:
- 红色警报闪烁
- OpenAI Logo（蓝色，左侧）vs Anthropic Logo（变暗，右侧）
- 五角大楼建筑 + "$1.4B" 金色数字炸裂

**字幕**: "AI圈出大事了！OpenAI刚拿到美军14亿美元大单"

**BGM**: 警报音效 + 爆炸音

**制作方法**:
1. 使用 Canva/剪映：搜索"科技背景"
2. 添加 OpenAI 和 Anthropic Logo
3. 使用"数字炸裂"特效

## 🛠️ 快速制作步骤

### 方法1: 使用剪映（推荐）⭐⭐⭐⭐⭐

1. **下载剪映 CAPCUT**
   - 官网: https://www.capcut.cn

2. **导入分镜脚本**
   - 新建 9:16 竖版项目
   - 按照上述分镜逐个制作

3. **使用模板**
   - 搜索"科技感"、"新闻类"模板
   - 替换文字和图片

4. **添加特效**
   - "数字滚动"、"爆炸"、"分屏"
   - 调整特效时长匹配节奏

5. **添加字幕**
   - 使用"自动字幕"功能
   - 或手动添加字幕（推荐）

6. **导出视频**
   - 分辨率: 1080×1920
   - 帧率: 30fps
   - 码率: 推荐

### 方法2: 使用 Seedance 2.0（AI生成）⭐⭐⭐⭐

1. **访问 Seedance**
   - 网址: https://jimeng.jianying.com

2. **输入提示词**
   ```
   赛博朋克科技风格，50秒，9:16竖屏，AI军备竞赛主题

   【开头 0-3秒】
   红色警报闪烁，OpenAI蓝色Logo vs Anthropic暗色Logo对撞，五角大楼建筑+$1.4B金色数字粒子爆炸

   【角度1 3-16秒】
   $1.4B数字快速滚动，¥10B人民币钞票雨，AI士兵对峙能量剑，"AI ARMS RACE"军事标题炸裂

   【角度2 16-30秒】
   AI大脑分裂科技vs伦理，Anthropic护盾+"NO"红色，五角大楼机械臂+"MUST COMPLY"，伦理vs军事天平摇摆

   【角度3 30-43秒】
   打工人剪影+?问号，AI机器人替代工作，AI失控红眼激光城市燃烧，"WHO'S RESPONSIBLE?"+深渊

   【结尾 43-50秒】
   分屏趋势蓝vs灾难红，评论区快速流动，关注按钮特写渐黑

   配乐: 赛博朋克电子乐BPM130-140，警报音效，科技音效
   字幕: 思源黑体Heavy，红色#FF3B0，描边+发光，数字黄色#FFD60A
   ```

3. **生成视频**
   - 点击"生成"
   - 等待 3-5 分钟

4. **下载和调整**
   - 下载生成的视频
   - 在剪映中微调
