## 🛡️ 安全防护三件套

### 1. Skill Vetter ⭐⭐⭐⭐⭐

**下载量**: 3,500+
**用途**: 安装前自动扫描代码

**功能**:
- ✅ 检查可疑的网络请求
- ✅ 检查文件读写操作
- ✅ 检查环境变量访问
- ✅ 相当于给 OpenClaw 装了安检门

**安装**:
```bash
npx clawhub@latest install skill-vetter
```

**使用**:
- 安装任何 Skill 之前，先运行 Skill Vetter
- 自动扫描代码，检查安全性
- 发现问题立即警告

---

### 2. Security Scanner ⭐⭐⭐⭐⭐

**用途**: 给每个 Skill 打安全评级

**评级标准**:
- 🟢 **SAFE** - 安全，可以使用
- 🟡 **CAUTION** - 需要注意权限
- 🔴 **DANGEROUS** - 危险，直接卸载

**使用规则**:
- 🔴 红灯 → 直接卸载
- 🟡 黄灯 → 仔细看权限申请再决定
- 🟢 绿灯 → 可以使用

**安装**:
```bash
npx clawhub@latest install security-scanner
```

---

### 3. 100/3法则 ⭐⭐⭐⭐⭐

**规则**: 只安装满足以下条件的 Skill：

1. **下载量**: 100+ 次
2. **发布历史**: 3个月以上

**原因**:
- 下载量少 → 可能未经过充分测试
- 发布时间短 → 可能有未知问题
- 质量不确定 → 可能存在安全风险

**判断标准**:
```
下载量 >= 100? → 否 → 不安装
发布时间 >= 3个月? → 否 → 不安装
都满足? → 是 → 继续安全检查
```

## 🔍 安装前检查流程

### 标准流程（严格执行）

```
发现新 Skill
  ↓
Step 1: 检查下载量和发布时间（100/3法则）
  ↓ 不满足 → 拒绝安装
  ↓ 满足
  ↓
Step 2: 运行 Skill Vetter 扫描代码
  ↓ 发现可疑代码 → 拒绝安装
  ↓ 代码安全
  ↓
Step 3: 运行 Security Scanner 评级
  ↓ 🔴 DANGEROUS → 拒绝安装
  ↓ 🟡 CAUTION → 人工审核
  ↓ 🟢 SAFE → 安装
  ↓
安装成功
```

## ⚠️ 安全原则

### 原则 1: 安全第一

**宁可错过好技能，不要安装危险技能**

- ✅ 安全优先
- ❌ 不冒险

### 原则 2: 自动化检查

**不要人工检查，用工具检查**

- ✅ Skill Vetter 自动扫描
- ✅ Security Scanner 自动评级
- ❌ 不要人工审查代码

### 原则 3: 严格筛选

**严格执行 100/3 法则**

- ✅ 下载量 >= 100
- ✅ 发布时间 >= 3个月
- ❌ 不满足不安装

## 💡 关键洞察

### 洞察 1: 安装前检查 > 安装后后悔

**问题**: 安装了危险 Skill → 安全风险
**解决**: 安装前严格检查 → 拒绝危险 Skill

### 洞察 2: 自动化 > 人工

**问题**: 人工检查容易遗漏
**解决**: 工具自动检查 → 全面覆盖

### 洞察 3: 严格筛选 > 追求新功能

**问题**: 安装了很多 Skill → 安全风险高
**解决**: 只安装安全 Skill → 质量优于数量
