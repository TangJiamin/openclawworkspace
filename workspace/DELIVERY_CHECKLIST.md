## 🎬 导出为 MP4 的 4 种方法

### 方法1: FFmpeg（推荐）⭐⭐⭐⭐⭐
```bash
ffmpeg -f concat -safe 0 -i video_frames.txt \
  -c:v libx264 -preset fast -crf 23 \
  -r 30 -pix_fmt yuv420p \
  output.mp4
```

### 方法2: 在线工具 ⭐⭐⭐⭐
- [EZGIF](https://ezgif.com/png-to-mp4)
- [CloudConvert](https://cloudconvert.com/png-to-mp4)
- [Convertio](https://convertio.co/zh/png-mp4/)

### 方法3: 视频编辑软件 ⭐⭐⭐⭐⭐
- Adobe Premiere Pro
- Final Cut Pro
- DaVinci Resolve
- 剪映（CapCut）

### 方法4: 浏览器预览录制 ⭐⭐⭐
1. 打开 `video_preview.html`
2. 使用 OBS 或 Camtasia 录制
3. 保存为 MP4
