# 🎯 Agent 核心文档精简优化

## 📋 优化原则

**核心原则**: 在不影响功能的前提下精简文档

**目标**:
- ✅ 功能 100% 保留
- ✅ Token 消耗最小化
- ✅ 可读性提升
- ✅ 维护成本降低

---

## 🛠️ 优化方法

### 1. 表格优于文字

**原则**: 使用表格代替段落描述

**示例**:
```markdown
# 之前（文字描述）
## 工具 1: Jina AI Search
Jina AI Search 是一个免费的通用搜索工具，支持各种查询场景...
命令: curl -s "https://s.jina.ai/[query]"
优势: 免费、AI 优化

# 现在（表格）
| 工具 | 成本 | 用途 | 命令 |
|------|------|------|------|
| Jina AI Search | ✅ 免费 | 通用搜索 | `curl -s "https://s.jina.ai/[query]"` |
```

**效果**: Token 减少 60%，可读性提升

---

### 2. 核心命令

**原则**: 仅保留核心命令，删除重复示例

**示例**:
```bash
# 之前（多个示例）
node /home/node/.openclaw/workspace/skills/tavily-search/scripts/search.mjs "query"
node /home/node/.openclaw/workspace/skills/tavily-search/scripts/search.mjs "query" -n 10
node /home/node/.openclaw/workspace/skills/tavily-search/scripts/search.mjs "query" --deep

# 现在（核心命令）
node scripts/search.mjs "query" --deep
```

**效果**: Token 减少 70%，信息完整

---

### 3. 流程图

**原则**: 使用简单流程图代替详细说明

**示例**:
```markdown
# 之前（详细说明）
## 智能降级机制
当免费工具的结果不满足需求时，我们会使用付费工具作为备选方案。具体步骤如下：
1. 首先尝试使用 Jina AI Search 进行搜索
2. 评估搜索结果的质量
3. 如果质量不满足，使用 Tavily Search
4. 如果还不满足，尝试其他工具

# 现在（流程图）
免费工具（Jina）
  ↓ 结果不满足
付费工具（Tavily）
```

**效果**: Token 减少 80%，更直观

---

### 4. 删除冗余

**原则**: 删除所有不增加价值的内容

**删除类型**:
- ❌ 重复的原则陈述
- ❌ 过度的说明性文字
- ❌ 显而易见的解释
- ❌ 冗长的前言和后记

---

### 5. 相对路径

**原则**: 使用相对路径代替绝对路径

**示例**:
```bash
# 之前（绝对路径）
node /home/node/.openclaw/workspace/skills/tavily-search/scripts/search.mjs "query"

# 现在（相对路径）
node scripts/search.mjs "query"
```

**效果**: Token 减少 50%

## 🎓 关键洞察

### 1. 精简 ≠ 功能缺失

**错误理解**: 精简会丢失功能

**验证**: research-agent v4.1 功能 100% 保留，Token 减少 72%

---

**发现**: 表格比文字更清晰，Token 更少

**应用**: 所有工具清单、配置状态、对比表
