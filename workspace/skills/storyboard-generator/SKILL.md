# 🎬 Storyboard Generator v1.0 - 分镜表生成器

将剧本转换为分镜表（Shot List）和静帧提示词，输出可直接用于图像生成的标准化数据（v1.0）。

## 🎯 核心能力

### 1. 剧本分析

**功能**: 深度分析剧本结构
- 提取本集信息（目标/主情绪/场景/角色）
- 识别场次划分
- 提取关键情节和冲突

### 2. 导演三帧分镜法

**核心思想**: 每场生成 3 张关键帧

- **起帧 (KF_A)**: 建立环境与关系
- **爆帧 (KF_B)**: 冲突/动作爆点
- **钩帧 (KF_C)**: 悬念/转场/情绪余波

### 3. 镜头表生成

**功能**: 展开为完整的镜头表

**格式**: ShotID, SceneID, 景别, 机位, 运镜, 目的, 唯一信息点, 定格瞬间, 动作链, 台词, CharID, SceneKey, 优先级, 状态

### 4. 静帧提示词生成

**三段式提示词**:
- **风格段**: STYLE_BASE 一句话
- **主体段**: @角色ID + 形象描述 + 表情动作
- **画面段**: SceneKey + 景别 + 机位 + 构图 + 定格瞬间

### 5. QC 检查体系

**三级优先级**:
- **P0**: 必须修复（影响剪辑）
- **P1**: 第二优先（影响一致性）
- **P2**: 最后修复（美观问题）

## 🎓 决策标准（AI 模型决策）

### 场景识别标准

当用户请求生成分镜表时，AI 模型应该根据以下标准判断：

#### 剧本类型识别

| 剧本类型 | 特征 | 处理方式 |
|---------|------|---------|
| 标准剧本 | 场标题、人物表、镜头动作 | 标准处理 |
| 文字描述 | 无场标题、无角色表 | 提取隐式场 |
| 大纲/概要 | 只有故事梗概 | 引导补充 |

#### 场次密度评估

| 时长 | 推荐密度 | 理由 |
|------|---------|------|
| 5-10 分钟 | 6-8 场 | 节奏紧凑 |
| 10-20 分钟 | 8-12 场 | 正常叙事 |
| 20-30 分钟 | 12-18 场 | 复杂叙事 |

## 🔄 工作流程

```
用户输入剧本
    ↓
步骤一：剧本分析
    ├─ 识别格式（标准/文字/大纲）
    ├─ 提取关键信息
    └─ 划分场次
    ↓
步骤二：导演三帧分镜（每场）
    ├─ 起帧：建立环境
    ├─ 爆帧：冲突爆点
    └─ 钩帧：悬念钩子
    ↓
步骤三：展开镜头表
    ├─ 基于三帧法展开
    ├─ 18 镜头/集（默认）
    ├─ 完整镜头序列
    └─ 详细的镜头描述
    ↓
步骤四：生成提示词
    ├─ 三段式提示词
    ├─ 引用资产库
    └─ 标准化输出
    ↓
步骤五：QC 检查
    ├─ P0: 必须修复
    ├─ P1: 第二优先
    └─ P2: 最后修复
    ↓
输出: 分镜表 + 提示词清单
```

## 🎭 景别选择指南

### 景别定义

| 景别 | 作用 | 示例场景 |
|------|------|----------|
| **远景**（Long Shot） | 建立环境、展示规模 | 山川全貌、城池远眺 |
| **全景**（Full Shot） | 展示人物全身、空间关系 | 角色站立、多人场景 |
| **中景**（Medium Shot） | 展示动作、人物互动 | 对话、交手 |
| **近景**（Close-up） | 展示表情、情绪 | 反应、独白 |
| **特写**（Extreme Close-up） | 强调细节、关键道具 | 眼神、手部、道具 |

### 机位选择指南

| 机位 | 情绪效果 | 示例场景 |
|------|----------|----------|
| **平视** | 中性、客观 | 日常对话 |
| **俯视** | 弱势、渺小 | 角色陷入困境 |
| **仰视** | 强势、威严 | 强者出场、压迫感 |
| **越肩**（Over Shoulder） | 代入、对峙 | 两人对话 |
| **主观**（POV） | 沉浸、紧张 | 发现敌人、探索 |

## 🎨 三段式提示词结构

### 强制格式

```
最终提示词 = 风格段 + 主体段 + 画面段
```

### 示例

**输入**:
- STYLE_BASE: "二次元国风仙侠风格"
- @角色ID: @jl_chenfeng
- SceneKey: scene_jl_yaoshugugu
- 景别: 近景
- 机位: 平视
- 定格瞬间: 陈风猛然转身，眼神警觉

**输出**:
```
二次元国风仙侠风格，陈风形象，破旧衣服，表情警觉，破旧小屋内，木质房梁，近景，平视，孤寂氛围
```

## ✅ 质量检查

### P0（必须修复）

- [ ] 是否有建立/反应/转场镜头？
- [ ] 定格瞬间是否"一眼看懂"？
- [ ] 是否有轴线跳跃？
- [ ] 信息点是否清晰？

### P1（第二优先）

- [ ] 角色脸/服装是否漂移？
- [ ] 道具是否与角色卡一致？
- [ ] 场景锚点是否保持？

### P2（最后修复）

- [ ] 画面是否足够干净？
- [ ] 光影是否统一？
- [ ] 是否足够电影化？
