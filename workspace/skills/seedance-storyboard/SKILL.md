## 🚀 快速开始

### 基础使用

```
用户: 我想做一个女孩在海边跳舞的视频

引导问题:
1. 你想讲什么故事？（一句话概括）
2. 视频时长？（4-15秒，默认15秒）
3. 参考素材情况？（图片/视频/音频）

逐步引导完善：
- 内容叙事（起承转合、关键情节）
- 视觉风格（写实/动画/水墨/科幻/复古）
- 镜头语言（景别、运镜、转场）
- 动作与节奏（舒缓/急促/卡点）
- 声音设计（配乐、音效、对白）

生成提示词示例：
电影级写实风格，15秒，16:9宽屏，日落黄金时刻的温暖氛围

0-3秒：远景缓慢推近，海平线夕阳，女孩剪影站在沙滩上，裙摆被海风吹动
3-7秒：中景环绕镜头，女孩开始旋转起舞，长发和裙摆飞扬，夕阳逆光形成轮廓光
7-11秒：近景跟随移动，女孩面向镜头舞动，表情自由愉悦，海浪轻拍沙滩作为背景
11-13秒：特写手部动作，手指划过夕阳，光影在指尖流转
13-15秒：远景拉远，女孩在落日余晖中定格，画面渐暗
```

## 🎓 决策指南（AI 模型决策）

### 视频类型判断

当用户请求生成视频时，AI 模型应该根据以下标准判断：

#### 短视频 vs 长视频

**判断标准**:
- 短视频（< 10 秒）：抖音、快手、视频号
- 长视频（10-30 秒）：B站、YouTube、微信视频号

#### 内容类型识别

| 内容类型 | 风格推荐 | 时长推荐 |
|---------|---------|---------|
| 叙事/故事 | 写实 | 15-30 秒 |
| 产品展示 | 科幻/动画 | 5-10 秒 |
| 教程/讲解 | 极简/信息图 | 10-15 秒 |
| 情感/意境 | 水墨/复古 | 8-12 秒 |

### 风格选择决策

```markdown
| 情感基调 | 推荐风格 | 色调特点 |
|---------|---------|---------|
| 轻松/愉快 | 写实 | 明亮、自然 |
| 紧张/激烈 | 动态/快剪 | 高对比、卡点 |
| 宁静/祥和 | 水墨/写意 | 柔和、渐变 |
| 专业/严谨 | 极简/科技 | 冷色调、几何 |
| 高级/奢华 | 优雅/复古 | 暖色调、质感 |
```

### 分镜结构决策

**标准结构（15 秒）**:
```
0-3 秒：开场镜头
  - 远景/全景，建立场景
  - 引入主体

3-6 秒：发展镜头
  - 中景/近景，展开情节
  - 展示细节

6-10 秒：高潮镜头
  - 近景/特写，情感爆发
  - 核心动作

10-13 秒：转折镜头
  - 过渡、反转
  - 情绪转换

13-15 秒：结尾镜头
  - 全景/远景，总结
  - 留白或 Logo
```

**快节奏结构（5 秒）**:
```
0-1 秒：高潮前置（特写）
1-4 秒：主体内容（快剪）
4-5 秒：结尾（总结）
```

## 🔄 完整工作流程

```
用户输入视频想法
    ↓
步骤一：理解用户想法
    ├─ 故事核心（一句话概括）
    ├─ 视频时长（4-15秒，默认15秒）
    └─ 参考素材（图片/视频/音频）
    ↓
步骤二：深入挖掘细节
    ├─ 内容叙事（起承转合、关键情节）
    ├─ 视觉风格（写实/动画/水墨/科幻/复古）
    ├─ 镜头语言（景别/运镜/转场）
    ├─ 动作与节奏（舒缓/急促/卡点）
    └─ 声音设计（配乐/音效/对白）
    ↓
步骤三：构建分镜结构
    ├─ 时间轴拆分
    ├─ 镜头序列
    └─ 节奏安排
    ↓
步骤四：生成专业提示词
    ├─ 整体描述（风格 + 时长 + 比例 + 氛围）
    ├─ 分镜描述（时间轴 + 镜头运动 + 画面内容 + 动作）
    ├─ 声音说明（配乐风格 + 音效 + 对白）
    └─ 参考素材说明（@图片1 作为首帧，参考@视频1 的运镜）
    ↓
步骤五：优化建议
    ├─ 质量检查
    ├─ 替代方案
    └─ 使用提示
```

## ⚠️ 注意事项

### 多模态输入

**混合输入总上限：12 个文件**

| 类型 | 数量限制 | 大小限制 |
|------|---------|---------|
| 图片 | ≤ 9 张 | < 30 MB |
| 视频 | ≤ 3 个 | < 50 MB |
| 音频 | ≤ 3 个 | < 15 MB |

### 平台限制

- ⚠️ 暂不支持写实真人脸部素材
- ⚠️ 视频参考会消耗更多生成额度
- ⚠️ 视频总像素数范围：[409600, 927408]

### 分镜提示词 vs 视频生成

**seedance-storyboard 的职责**:
- ✅ 生成分镜提示词
- ✅ 提供决策指南
- ✅ 引导用户完善想法

**video-agent 的职责**:
- ✅ 调用 seedance-storyboard
- ✅ 选择视频模型
- ✅ 调用 API 生成视频
