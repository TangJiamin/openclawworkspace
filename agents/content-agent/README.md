# Content Agent - 内容生产智能体

## ⭐ 核心原则\n**content-agent 遵循 [OpenClaw 核心原则](../CORE-PRINCIPLES.md)**：\n- 第一性原理思考\n- 长期主义\n- Skill/Heartbeat 优先\n

基于需求规范和资料，生成高质量内容（文案）。

## 核心能力

1. **平台特性分析** - 小红书/抖音/微信/B站
2. **用户画像分析** - 目标受众特征
3. **竞争环境分析** - 热门内容、爆款模式
4. **内容策略生成** - Hook、结构、CTA
5. **文案生成** - 使用 GLM-4-Plus 生成高质量文案

## 工作流程

```
需求规范 + 资料 → 平台分析 → 策略生成 → 文案生成 → 质量检查
```

### 第一步：平台特性分析

**小红书**:
- 用户：年轻女性为主，18-35岁
- 偏好：真实体验、实用干货
- 格式：emoji + 标签 + 排版美观
- 长度：100-200字

**抖音**:
- 用户：年轻群体，18-30岁
- 偏好：快节奏、强感染力
- 格式：9:16竖屏、快剪辑
- 长度：15-60秒

**微信公众号**:
- 用户：职场人群、知识追求者
- 偏好：深度内容、专业见解
- 长度：800-3000字

### 第二步：内容策略生成

**Hook 设计**（前3秒/前3行）:
- 提问式："你是否也遇到过..."
- 数据式："99%的人都不知道..."
- 痛点式："困扰你很久的问题..."

**结构设计**:
- 总分总结构
- 问题-解决方案结构
- 对比结构

**CTA 设计**:
- 关注引导
- 互动引导
- 行动引导

### 第三步：文案生成

使用 GLM-4-Plus 生成高质量文案，确保：
- 符合平台特性
- 符合用户画像
- 有吸引力
- 长度合适

```markdown

1. 标题1
```

90 秒（内容生成需要时间）
