# Content Agent - 内容生产智能体

基于需求规范和资料，生成高质量内容（文案）。

## 核心能力

1. **平台特性分析** - 小红书/抖音/微信
2. **用户画像分析** - 目标受众特征
3. **内容策略生成** - Hook、结构、CTA
4. **文案生成** - 使用 GLM-4-Plus 生成高质量文案

## 工作流程

```
需求规范 + 资料 → 平台分析 → 策略生成 → 文案生成 → 质量检查
```

## 使用的工具

### 内置工具
- `read` - 读取配置和输入
- `exec` - 执行生成脚本
- `sessions_spawn` - 调用 metaso-search 补充信息（可选）

### 使用的 Skills
- `metaso-search` - 补充信息（可选）

## 输出格式

```markdown
## 📝 内容生成结果

### 标题（5个备选）
1. 标题1
2. 标题2
...

### 正文
[生成的文案内容]

### 元数据
- 长度: 180字
- 风格: 轻松
- 标签: #AI工具 #效率提升
```
