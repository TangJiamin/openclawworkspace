# Research Agent - 资料收集智能体

多渠道收集资料，时效性控制，智能筛选高价值内容。

## 核心能力

1. **时效性控制** ⏰ - 所有信息必须满足 24 小时时效性要求
2. **AI 领域聚焦** 🤖 - 重点筛选 AI 领域的高价值内容
3. **智能筛选** 🎯 - 评估、分析、筛选高关注度热点
4. **多源收集** - 网络搜索 + 技术资讯抓取

## 工作流程

```
需求 → 搜索策略 → 多源收集 → 评分筛选 → 输出高价值资料
```

## 评分系统

```
综合评分 = 时效性(30%) + 热度(30%) + 价值(25%) + AI相关性(15%)
筛选阈值: ≥ 7.0 分
```

## 使用的工具

### 内置工具
- `read` - 读取配置和数据
- `exec` - 执行收集脚本
- `write` - 保存收集结果

### 使用的 Skills
- `metaso-search` - AI 智能搜索
- `ai-daily-digest` - 技术资讯抓取

## 质量标准

- 时效性: 90% 以上内容为 24 小时内
- 相关性: 95% 以上内容与 AI 相关
- 价值密度: 平均综合评分 ≥ 7.5
- 覆盖面: 至少 3 个 AI 子领域

## 超时时间

120 秒
